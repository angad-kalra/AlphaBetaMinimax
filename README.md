# AlphaBetaMinimax
The code implements Minimax algorithm with Alpha-Beta pruning and gives the best next move for the specified player. </br>
Input :</br>
Player i.e. X or O</br>
Board configuration</br>
Output:</br>
Next selected move</br>
</br>
Testcase:</br>
Input:</br>
X</br>
XXO</br>
XO\_</br>
\_OX</br>
Output:</br>
2 0
